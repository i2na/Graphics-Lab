# Computer Graphics Projects

컴퓨터 그래픽스 실습 프로젝트 모음입니다. Three.js와 WebGPU를 사용한 인터랙티브 3D 그래픽스 구현이 포함되어 있습니다.

### [`Lamp`](https://github.com/i2na/Graphics-Lab/tree/main/lamp)

- **설명**: Three.js로 구현한 Lamp 인터랙티브 시뮬레이션
- **데모**: [https://i2na.github.io/Graphics-Lab/lamp](https://i2na.github.io/Graphics-Lab/lamp)
- **기술**: Three.js, WebGL

### [`Hand`](https://github.com/i2na/Graphics-Lab/tree/main/hand)

- **설명**: Three.js를 사용하여 인터랙티브한 손 모델 구현
- **데모**: [https://i2na.github.io/Graphics-Lab/hand](https://i2na.github.io/Graphics-Lab/hand)
- **기술**: Three.js, WebGL

### [`Tank`](https://github.com/i2na/Graphics-Lab/tree/main/tank)

- **설명**: WebGPU 기반 탱크 씬 렌더링과 간단한 인터랙션
- **데모**: [https://i2na.github.io/Graphics-Lab/tank](https://i2na.github.io/Graphics-Lab/tank)
- **기술**: WebGPU, WGSL, PBR Shading
